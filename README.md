# demo_TelegramBot_NFT
This project is a Telegram web application that allows users to buy and sell NFTs. It was created by a developer with the purpose of trying to create a telegram webapp for product purchases. In this application users can add products to the cart, order products, and pay for them.

## Installation:
1. To install the dependencies for this project, run the following command: `pip install -r requirements.txt`
2. To run this project, run the following command: `python main.py`
